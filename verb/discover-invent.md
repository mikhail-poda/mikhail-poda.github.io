# Topic: to Discover, Invent

| **Infinitive**   | **Transcription** | **Meaning**                               |
| ---------------- | ----------------- | ----------------------------------------- |
| **לְגַלּוֹת**       | *legalót*         | to discover, find out; to reveal       |
| **לְהַמְצִיא**      | *lehamtsí*        | to invent; (also) to make up              |
| **לְאַתֵּר**        | *le’atér*         | to locate, identify, trace          |
| **לְחַשּׂוֹף**       | *lechasóf*        | to expose, uncover, reveal publicly |
| **לְחַדֵּשׁ**        | *lechadésh*       | to innovate; to renew, update             |
| **לְפַתֵּחַ**        | *lefatéach*       | to develop (something)                    |
| **לְהִתְפַּתֵּחַ**      | *lehitpatéach*    | to develop, evolve (by itself)         |
| **לִפְתּוֹר**       | *liftór*          | to solve, resolve                      |
