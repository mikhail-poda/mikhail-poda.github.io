# Hebrew Numerals

## One 

The “one” is special - it behaves like an adjective, following the noun like **סֵפֶר אֶחָד**.

Forms: **אֶחָד** (m), **אַחַת** (f).

| **Hebrew**  | **Transcription** | **Meaning**         | 
| ----------- | ----------------- | ------------------- | 
| **לְהִתְאַחֵד**  | *lechit'ached*   | to unite, come together | 
| **לְאַחֵד**    | *leached*        | to unite, conolidate  | 
| **הִתְאַחֲדוּת** | *hit'achadut*    | association          | 
| **אַחְדוּת**   | *achdut*         | unity                | 
| **אִיחוּד**   | *ichud*          | unity, unification   | 
| **אָחִיד**    | *achid*          | uniform, homogeneous | 
| **מְאוּחָד**   | *me'uchad*       | united               |
