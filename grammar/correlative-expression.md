# Topic: Conditional and Correlative Expressions

## Conditional Expressions

| **Pattern**                  | **Meaning**                           | **Example**                                                                                |
| ---------------------------- | ------------------------------------- | ------------------------------------------------------------------------------------------ |
| &lrm;**לָמָּה …? כִּי …**‏              | why …? because …                      | **למה אתה עייף? כי לא ישנתי.** – Why are you tired? Because I didn’t sleep.                |
| &lrm;**מַדוּעַ …? מִפְּנֵי שֶׁ־ …**‏        | why …? because … (more formal)        | **מדוע הוא כועס? מפני שהוא חיכה הרבה.** – Why is he angry? Because he waited a lot.        |
| &lrm;**מָתַי …? כַּאֲשֶׁר …**‏            | when …? when/whenever … (more formal) | **מתי זה קורה? כאשר יורד גשם.** – When does it happen? When it rains.                      |
| &lrm;**מָתַי …? כְּשֶׁ־ …**‏             | when …? when … (common)               | **מתי נצא? כשאתה מוכן.** – When do we leave? When you’re ready.                            |

## Correlative Expressions

| **Pattern**                 | **Meaning**                           | **Example**                                                                                |
| --------------------------- | ------------------------------------- | ------------------------------------------------------------------------------------------ |
| &lrm;**אִם … אָז …**‏                | if … then …                           | **אם תרצה, אז נעשה את זה.** – If you want, then we’ll do it.                               |
| &lrm;**מ־… עַד …**‏                 | from … until/to …                     | **מ־8 עד 10 אני בעבודה.** – From 8 to 10 I’m at work.                                      |
| &lrm;**מ־… ל־ …**‏                 | from … to … (change/range)            | **זה עלה מ־50 ל־70.** – It went from 50 to 70.                                             |
| &lrm;**לֹא … אַף …**‏                | not … any / not even one              | **לא ראיתי אף אחד.** – I didn’t see anyone.                                                |
| &lrm;**אֵין … אַף …**‏               | there isn’t … any                     | **אין לי אף רעיון.** – I don’t have any idea.                                              |
| &lrm;**כְּדֵי … כְּדַאי …**‏             | in order to …, it’s advisable to …    | **כדי להצליח, כדאי להתאמן.** – To succeed, it’s good to practice.                          |
| &lrm;**בֵּין … וּבֵין …**‏             | between … and …                       | **בין הבית ובין העבודה יש פארק.** – Between home and work there’s a park.                  |
| &lrm;**בֵּין … לְ־ …**‏               | between … and …                       | **בין 3 ל־4 אני פנוי.** – Between 3 and 4 I’m free.                                        |
| &lrm;**או … או …**‏                | either … or …                         | **או אתה או אני.** – Either you or me.                                                     |
| &lrm;**לֹא … וְלֹא …**‏               | neither … nor …                       | **לא קפה ולא תה.** – Neither coffee nor tea.                                               |
| &lrm;**לֹא רַק … אֶלָא גַּם …**‏         | not only … but also …                 | **לא רק הוא בא, אלא גם היא.** – Not only he came, but she also did.                        |
| &lrm;**לֹא … אֶלָא …**‏               | not … but rather …                    | **זה לא יקר, אלא סביר.** – It’s not expensive, but rather reasonable.                      |
| &lrm;**גַּם … וְגַם …**‏               | both … and …                          | **גם הוא וגם היא באו.** – Both he and she came.                                             |
| &lrm;**כְּמוֹ … כָּךְ …**‏               | just as … so …                        | **כמו שהוא עוזר לי, כך אני עוזר לו.** – Just as he helps me, so I help him.                 |
| &lrm;**בֵּין אִם … או …**‏            | whether … or …                        | **בין אם תבוא או לא—נתחיל.** – Whether you come or not, we’ll start.                       |
| &lrm;**בֵּין אִם … וּבֵין אִם …**‏       | whether … or whether …                | **בין אם זה נכון ובין אם לא—נבדוק.** – Whether it’s true or not, we’ll check.              |
| &lrm;**כְּכָל שֶׁ־ … כָּךְ …**‏            | the more … the more …                 | **ככל שאתה לומד, כך אתה יודע יותר.** – The more you learn, the more you know.              |
| &lrm;**לֹא פָּחוֹת … מֵאֲשֶׁר …**‏         | no less … than …                      | **זה לא פחות חשוב מאשר כסף.** – This is no less important than money.                      |
| &lrm;**קוֹדֶם …, אַחַר כָּךְ …**‏         | first …, then …                       | **קודם נאכל, אחר כך נצא.** – First we’ll eat, then we’ll go out.                           |
| &lrm;**לַמְרוֹת שֶׁ־ … בְּכָל זֹאת …**‏     | even though … nevertheless …          | **למרות שהיה קשה, בכל זאת הצלחנו.** – Even though it was hard, nevertheless we succeeded.  |
| &lrm;**מִצַּד אֶחָד … מִצַּד שֵׁנִי …**‏      | on the one hand … on the other hand … | **מצד אחד זה יקר, מצד שני זה איכותי.** – On the one hand it’s expensive; on the other hand it’s high quality. |
| &lrm;**בִּכְלָל … בִּפְרָט …**‏            | in general … in particular …          | **בכלל זה מעניין, ובפרט ההתחלה.** – In general it’s interesting, especially the beginning. |
