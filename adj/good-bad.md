# Topic: Good and Bad

| **Positive** | **Transcription** | **Negative** | **Transcription** | **Main meaning**             | **Example**                                                                                                       |
| ------------ | ----------------- | ------------ | ----------------- | ---------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **טוֹב**      | *tov*             | **רַע**       | *ra*              | good / bad                   | **זה רעיון טוב / רע.** (*ze ra’ayon tov / ra*) – This is a good / bad idea.                                       |
| **אֵיכוּתִי**   | *eikhuti*         | **גָּרוּעַ**     | *garu’a*          | high-quality / poor-quality  | **זה מוצר איכותי / גרוע.** (*ze mutzar eikhuti / garu’a*) – This is a high-quality / poor-quality product.        |
| **מְצוּיָּן**    | *metzuyan*        | **גָּרוּעַ**     | *garu’a*          | excellent / poor             | **השירות מצוין / גרוע.** (*ha-sherut metzuyan / garu’a*) – The service is excellent / poor.                       |
| **נֶהְדָּר**     | *nehdar*          | **נוֹרָא**     | *nora*            | wonderful / terrible          | **היה לנו יום נהדר / נורא.** (*haya lanu yom nehdar / nora*) – We had a wonderful / terrible day.                 |
| **נִפְלָא**     | *nifla*           | **נוֹרָא**     | *nora*            | marvelous / terrible        | **זו חוויה נפלאה / נוראה.** (*zo chava’ya nifla’a / nora’a*) – This is a wonderful / terrible experience.         |
| **מוֹעִיל**    | *mo’il*           | **מַזִּיק**     | *mazik*           | useful, beneficial / harmful | **העצה הזאת מועילה / מזיקה.** (*ha-etsa ha-zot mo’ila / mazika*) – This advice is useful / harmful.               |
| **חִיּוּבִי**    | *chiyuvi*         | **שְׁלִילִי**    | *shlili*          | positive / negative          | **זו התפתחות חיובית / שלילית.** (*zo hitpatchut chiyuvit / shlilit*) – This is a positive / negative development. |

## Notes:

- **רַע** general “bad”
- **גָּרוּעַ**  “bad in quality, poor, lousy”
- **נוֹרָא**  “terrible, awful,” more emotional and intense
